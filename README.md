# SkeletonBowChance

SkeletonBowChance is a simple mod that adds a configurable chance for skeletons to spawn with bows.

I made this mod because skeletons are disproportionately strong compared to other mobs. This mod is designed to bring them to where you feel they are balanced.

## Requirements
- neoforge 1.21.1

## Usage
In Game:

Mods -> SkeletonBowChance -> Config
Range 0-100 (default 50)
